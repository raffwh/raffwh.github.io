# HUGO THEME: STACK
Source:   [STACK main repo](https://github.com/CaiJimmy/hugo-theme-stack)


# QUICK REMINDER ON HOW THIS WAS INSTALLED 
1. Go to https://github.com/CaiJimmy/hugo-theme-stack-starter
1. Click `use this template` in the top right corner
1. Create new repo
1. In the new repo for the github space, create a codespace
    1. Edit config/_default/config.toml
        - change baseurl to the github io code, including the final slash.
    1. In the VS Codespace terminal, run `hugo server`

1. Back to repo, go to setting>pages
    1. change the build deployment branch into **gh-pages**
